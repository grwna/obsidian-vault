# Layered Protocol
Almost all traffic uses IP (Internet Protocol) on the *network layer*, but on top of this they can use TCP and UDP (transport layer), and on top of all this use *application layer* like HTTP, FTP, or SMTP.

## Network Protocols
- IP
- ARP
## Transport Protocols
- TCP
- UDP
- QUIC
## Application
- HTTP/S
- FTP
